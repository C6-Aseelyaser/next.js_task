## B : Installing:

- Clone the repo to your local machine using git bash. ~~>https://github.com/C6-Aseelyaser/next.js_task
- Setup the project

```
npm i create-next-app
```
- installs a package
```
npm i
```

- Run server using git bash inside folder ~~> 

```
npm run dev
```

- Installing packages
```
npm i next-auth
```
now the app is ready to use
